# Image Classification with Convolutional Neural Networks
Simple CNN implemented with keras, trains CIFAR10 dataset (60,000 images) and categorizes them into the following 10 categories:
* "airplane"
* "auto"
* "bird"
* "cat"
* "deer"
* "dog"
* "frog"
* "horse"
* "ship"
* "truck"
